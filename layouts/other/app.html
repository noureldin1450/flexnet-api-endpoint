Make browsing up to 50% faster and more private while utilizing your full device screen without the web browser status bar or other elements

Installation Instructions

  1. Load Flexnet.ga in the Chrome Web Browser.
  2. Tap the 3 dots from the top right corner.
  3. Select "Add to Home screen" from the available options.
  4. Chrome will open a prompt and you will be able to add FLEXNET app icon to your home screen. Launch the app once to ensure the install completes successfully.
